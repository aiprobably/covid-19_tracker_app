# covid-19_tracker_app

Requirements:
wheel
streamlit
pandas
numpy
requests
plotly

Procedure to run program:
Step 1: Git clone https://github.com/aiprobably/covid-19_tracker_app
cd covid-19_tracker_app
pip install <requirements>

For windows:-
Step 2: Open cmd 
streamlit run covid-track-app.py

for linux:-
$streamlit Hello
streamlit run covid-track-app.py
